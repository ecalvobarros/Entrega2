# Proyecto - Ejercicio 10

## Descripción

Este proyecto contiene la resolución del ejercicio 10, incluyendo el cálculo de estadísticas y generación de tablas de posiciones.

## Instalación

1. Crear entorno virtual:
   python -m venv venv

2. Activar entorno virtual:
   .\venv\Scripts\activate

3. Instalar dependencias:
   pip install -r requirements.txt

## Ejecución

1. Abrir Jupyter Notebook:
   jupyter notebook

2. Ir a la carpeta notebooks

3. Ejecutar el archivo .ipynb correspondiente

## Estructura

* src/: contiene las funciones del programa
* notebooks/: contiene los notebooks para ejecutar los ejercicios

## Notas

Las funciones están implementadas en la carpeta src y son utilizadas desde los notebooks.
